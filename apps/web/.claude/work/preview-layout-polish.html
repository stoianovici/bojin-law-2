Summary of Changes

Typography Scale

  • Nav text → 12px
  • Button sm → 11px
  • Button md/lg → 12px
  • Search box → 12px

Spacing

  • Nav padding → py-2.5
  • Search height → 32px
  • Nav icons → 18px

Visual Feedback

  • Active bg → 15%
  • Active border → 2px left
  • Logo tracking → tight
  • Logo separator → aligned
Note: These changes align the UI with the Linear design system's typography scale (11px-20px with 13px base) rather than Tailwind's default scale (12px-30px with 16px base). The result is a more refined, compact interface consistent with Linear's aesthetic.
