# OPS-326: Desktop Page Migration

**Status**: In Progress
**Priority**: P1-High
**Type**: Feature
**Platform**: Desktop
**Parent**: OPS-322
**Blocked by**: OPS-323

---

## Problem Statement

~45 pages still use old gray/blue styles (`bg-gray-50`, `bg-white`, `text-gray-*`, `bg-blue-600`) instead of Linear design tokens. Pages need systematic migration to use structural components and Linear tokens.

---

## Approach

### Migration Priority

**Priority 1 - High Traffic Pages:**
- [x] `/page.tsx` (Dashboard/Home) - fallback migrated
- [x] `/cases/page.tsx`
- [x] `/tasks/page.tsx`
- [x] `/documents/page.tsx`
- [x] `/login/page.tsx`

**Priority 2 - Secondary Pages:**
- [x] `/cases/[caseId]/page.tsx`
- [x] `/clients/[id]/page.tsx`
- [x] `/time-tracking/page.tsx`
- [x] `/analytics/page.tsx`
- [x] `/settings/billing/page.tsx` (main settings page)

**Priority 3 - Admin/Utility:**
- [x] `/admin/ai-ops/page.tsx` (main admin page migrated)
- [x] `/reviews/page.tsx` (main reviews page migrated)
- [ ] Remaining `/admin/*` sub-pages
- [ ] Remaining `/reviews/*` sub-pages
- [ ] Error pages (403, 404, etc.)

### Migration Checklist Per Page

1. **Background Colors**
   - `bg-gray-50` → `bg-linear-bg-primary`
   - `bg-white` → `bg-linear-bg-secondary`
   - `bg-gray-100` → `bg-linear-bg-tertiary`

2. **Text Colors**
   - `text-gray-900` → `text-linear-text-primary`
   - `text-gray-600/700` → `text-linear-text-secondary`
   - `text-gray-400/500` → `text-linear-text-tertiary`

3. **Accent Colors**
   - `bg-blue-600` → `bg-linear-accent`
   - `text-blue-600` → `text-linear-accent`

4. **Border Colors**
   - `border-gray-200` → `border-linear-subtle`
   - `border-gray-300` → `border-linear-default`

5. **Structural Components** (from OPS-323)
   - Wrap page content in `PageLayout`
   - Use `SectionHeader` for section titles
   - Use `ListItem` for list rows
   - Use `MetricCard` for metrics
   - Use `WidgetGrid` for card layouts
