# OPS-359: Tasks Page

**Type**: Feature | **Priority**: P2-Medium | **Platform**: Desktop | **Status**: Open
**Parent**: [OPS-338](ops-338.md) | **Phase**: 4 (Pages)
**Blocked by**: OPS-349, OPS-350, OPS-353

## Problem

Tasks page needs list view with collapsible sections, 2-column layout with fixed detail panel.

## Approach

- Create TasksListPage with 2-column layout
- Left: collapsible sections (Urgente, Această săptămână, Finalizate)
- Right: fixed detail panel (subtasks, activity, comments)
- Create NewTaskModal with form fields

## Done When

- [x] Page header with search, toggles, filters, "+ Sarcină nouă"
- [x] 2-column layout with fixed right panel
- [x] TasksListView: collapsible sections with task items
- [x] TaskDetailPanel: subtasks with progress, activity feed, comment input
- [x] NewTaskModal: title, description, case, priority, due date, assignee, subtasks
