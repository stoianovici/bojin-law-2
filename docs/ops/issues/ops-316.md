## Implementation Notes

Used CSS custom properties rather than hardcoded colors so themes can switch dynamically:

- Light theme: `:root` defines light mode values
- Dark theme: `.dark` class overrides with Linear's signature dark palette

Inter font is loaded via Next.js font optimization (not CDN) for better performance and no FOUC.
