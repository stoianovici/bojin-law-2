# OPS-329: Loading/Empty/Error States & Transitions

**Status**: New
**Priority**: P2-Medium
**Type**: Feature
**Platform**: Both
**Parent**: OPS-322
**Blocked by**: OPS-326, OPS-328

---

## Problem Statement

Loading skeletons, empty states, and error states don't match Linear dark mode style. Transitions and animations are inconsistent across the app.

---

## Approach

### 1. Loading States (Skeletons)

Update skeleton components to use Linear dark mode colors:

```css
/* Current (light mode) */
.skeleton {
  background: #e5e7eb; /* gray-200 */
}

/* Linear dark mode */
.skeleton {
  background: linear-gradient(
    90deg,
    var(--bg-tertiary) 0%,
    var(--bg-hover) 50%,
    var(--bg-tertiary) 100%
  );
  background-size: 200% 100%;
  animation: shimmer 1.5s infinite;
}

@keyframes shimmer {
  0% {
    background-position: 200% 0;
  }
  100% {
    background-position: -200% 0;
  }
}
```

### 2. Empty States

Create consistent `EmptyState` component:

```tsx
// apps/web/src/components/linear/EmptyState.tsx

interface EmptyStateProps {
  icon: React.ReactNode;
  title: string;
  description?: string;
  action?: {
    label: string;
    onClick: () => void;
  };
}

// Visual: Centered, muted icon, title, description, optional button
```

### 3. Error States

Create `ErrorBoundary` with Linear styling:

```tsx
// apps/web/src/components/linear/ErrorBoundary.tsx

// Catches errors in children
// Shows friendly error message
// Option to retry or go home
// Logs to error tracking
```

Inline error messages:

```css
.error-message {
  color: var(--status-error);
  font-size: 13px;
  margin-top: 4px;
}
```

### 4. Transitions

Standardize transition timing:

```css
/* Interactive elements (buttons, links) */
transition: all 0.15s ease;

/* Page transitions */
transition: opacity 0.2s ease;

/* List item entrance */
@keyframes fadeSlideIn {
  from {
    opacity: 0;
    transform: translateY(8px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
animation: fadeSlideIn 0.2s ease forwards;
```

### 5. Focus Management

```css
/* Focus rings */
:focus-visible {
  outline: 2px solid var(--accent-primary);
  outline-offset: 2px;
}

/* Focus trap for modals */
/* Skip links for accessibility */
```
