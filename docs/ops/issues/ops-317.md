## Goal

Restyle high-priority shared components with Linear dark mode design.

## Components

1. **Card** (`apps/web/src/components/ui/card.tsx`)
   - Dark background (`linear-bg-secondary`)
   - Subtle border (`linear-subtle`)
   - Hover effect with elevated border

2. **Button** (primary, secondary, ghost variants)
   - Primary: Accent color with glow
   - Secondary: Tertiary background
   - Ghost: Transparent with hover state

3. **Badge / Status indicators**
   - Success: Green with muted background
   - Warning: Amber with muted background
   - Error: Red with muted background
   - Info: Blue with muted background

4. **Input / Search box**
   - Tertiary background
   - Subtle border
   - Focus ring with accent color

5. **Progress bar**
   - Track: Tertiary background
   - Fill: Accent color

## Pattern

```tsx
// Before
className = 'bg-white border-gray-200 text-gray-900';

// After (dark mode)
className = 'bg-linear-bg-secondary border-linear-subtle text-linear-text-primary';
```
