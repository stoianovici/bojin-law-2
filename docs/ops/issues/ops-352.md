# OPS-352: Table Components

**Type**: Feature | **Priority**: P2-Medium | **Platform**: Desktop | **Status**: Open
**Parent**: [OPS-338](ops-338.md) | **Phase**: 2 (Composite)
**Blocked by**: Phase 1 (OPS-345, OPS-346, OPS-347, OPS-348)

## Problem

Three table patterns needed: MinimalTable (cases), GroupedTable (time entries), Report Table (analytics).

## Approach

- Create MinimalTable with light header, row hover, title+subtitle column
- Create GroupedTable with collapsible headers and counts
- Reuse for report tables with filter/export slots

## Done When

- [x] MinimalTable: light header, row hover, sortable columns
- [x] Title + subtitle pattern in first column
- [x] GroupedTable: collapsible group headers with counts
- [x] Date-based grouping support
- [x] Report table with totals footer
