# OPS-306: BriefRow Component

**Type**: Feature | **Priority**: P1-High | **Platform**: Mobile
**Status**: Completed
**Created**: 2024-12-28

## Problem

Current `BriefCard` uses card wrappers with shadows and borders, making each item feel like a discrete object. Need a typography-first component that feels like reading a paragraph in a document.

## Approach

Create `BriefRow.tsx` with:

- No card wrapper, just text layout
- Bold title (sender name or action)
- Normal weight preview text (1-2 lines)
- Small gray metadata line with case name + relative time
- Tree-style prefix (`└`) for metadata to create visual hierarchy
- Tappable area for navigation

```tsx
<button className="w-full text-left py-3 px-4 active:bg-gray-50">
  <div className="font-semibold text-gray-900">{title}</div>
  <div className="text-gray-600 text-sm line-clamp-2">{preview}</div>
  <div className="text-xs text-gray-400 mt-1 flex items-center gap-1">
    <span>└</span>
    <span>{caseName}</span>
    <span>·</span>
    <span>{relativeTime}</span>
  </div>
</button>
```
