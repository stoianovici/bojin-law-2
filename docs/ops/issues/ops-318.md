## Goal

Restyle secondary shared components with Linear dark mode design.

## Components

1. **List items** (task rows, case rows)
   - Tertiary background on hover
   - Subtle separator borders
   - Active state with accent tint

2. **Section headers**
   - Uppercase, small caps styling
   - Tertiary text color
   - Micro font size (11px / 600 weight)

3. **Avatar**
   - Border ring option
   - Fallback with gradient background

4. **Dropdown/Select**
   - Secondary background
   - Subtle border
   - Hover states for options

5. **Checkbox**
   - Tertiary background unchecked
   - Accent fill when checked
   - Focus ring

## Pattern

```tsx
// List item
className = 'hover:bg-linear-bg-hover border-b border-linear-subtle';

// Section header
className = 'text-linear-text-tertiary text-[11px] font-semibold uppercase tracking-wider';
```
