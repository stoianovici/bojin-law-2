# OPS-351: Document Card Component

**Type**: Feature | **Priority**: P2-Medium | **Platform**: Desktop | **Status**: Done
**Parent**: [OPS-338](ops-338.md) | **Phase**: 2 (Composite)
**Blocked by**: Phase 1 (OPS-345, OPS-346, OPS-347, OPS-348)

## Problem

Document cards need thumbnail, file type badge, size, action buttons (Mapă, Word). Selected state with dashed border.

## Approach

- Create DocumentCard with thumbnail slot
- Add file type badge (top-right, colored by type)
- Add action buttons at bottom
- Add selected state (dashed border)

## Done When

- [x] DocumentCard with thumbnail preview area
- [x] File type badge (PDF red, DOCX blue, XLSX green, IMG orange)
- [x] File info (name, size)
- [x] Action buttons (Mapă, Word)
- [x] Selected state with dashed accent border
