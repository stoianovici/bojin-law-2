# OPS-345: Button System Consolidation

**Type**: Feature | **Priority**: P2-Medium | **Platform**: Desktop | **Status**: Ready for Verification
**Parent**: [OPS-338](ops-338.md) | **Phase**: 1 (Foundation)

## Problem

Multiple button patterns exist (`packages/ui`, `apps/web/components/ui`). Need unified button system with all variants from design spec: primary, secondary, ghost, danger, icon-only, action buttons.

## Approach

- Consolidate to single Button component with CVA variants
- Add IconButton and ActionButton variants
- Ensure loading state with spinner
- Add keyboard shortcut hint support
