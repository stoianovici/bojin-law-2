## Goal

Restyle desktop-specific layout components with Linear dark mode design.

## Tasks

### 1. Sidebar Navigation

- Dark background (`#111113`)
- Subtle border right
- Active item with accent tint (`rgba(94, 106, 210, 0.15)`)
- Section titles uppercase, small caps
- Logo/branding area

### 2. Header Bar

- Blur backdrop (`backdrop-filter: blur(12px)`)
- Search box with `Cmd+K` shortcut hint
- Icon buttons with notification badges
- User menu

### 3. Dashboard Grid

- Card spacing and borders
- Widget containers
- Responsive column layout

### 4. Morning Briefing Card

- Gradient accent line at top
- Stats grid layout
- Priority tasks section

## CSS Patterns

```css
/* Sidebar */
.sidebar {
  background: #111113;
  border-right: 1px solid rgba(255, 255, 255, 0.06);
}

/* Active nav item */
.nav-item-active {
  background: rgba(94, 106, 210, 0.15);
  color: #5e6ad2;
}

/* Header blur */
.header {
  backdrop-filter: blur(12px);
  background: rgba(17, 17, 19, 0.8);
}
```
