# OPS-355: Toast Notification System

**Type**: Feature | **Priority**: P2-Medium | **Platform**: Desktop | **Status**: Open
**Parent**: [OPS-338](ops-338.md) | **Phase**: 3 (Modals)
**Parallel**: Yes

## Problem

Toast system exists but needs standardization: 380px width, bottom-right, variants, stacking.

## Approach

- Update Toast to 380px width
- Add icon + title + message pattern
- Add success/error/warning/info variants
- Implement auto-dismiss (5s) and stacking

## Done When

- [x] Toast: 380px width, bottom-right position
- [x] Icon + title + message layout
- [x] Variants: success, error, warning, info
- [x] Auto-dismiss after 5s (success/info), manual for error/warning
- [x] Stack multiple toasts (12px gap)
