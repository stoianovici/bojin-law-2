# OPS-361: Calendar Page

**Type**: Feature | **Priority**: P2-Medium | **Platform**: Desktop | **Status**: Verifying
**Parent**: [OPS-338](ops-338.md) | **Phase**: 5 (More Pages)
**Blocked by**: OPS-349, OPS-353

## Problem

Calendar needs week view (Mon-Fri), sidebar with type/team filters, event blocks with colors.

## Approach

- Create CalendarPage with sidebar filters
- Implement WeekView with 5-day columns and time grid
- Create MonthView and AgendaView
- Add event type colors from design spec

## Done When

- [x] Page header with "Eveniment nou", view toggle, navigation
- [x] Sidebar: calendar type filters, team member checkboxes
- [x] WeekView: Mon-Fri columns, hour grid, current time indicator
- [x] Event blocks with type-based colors
- [x] MonthView: calendar grid with event dots
- [x] AgendaView: date sections with event list
- [x] EventDetailModal, NewEventModal
- [x] DayView: single day with time grid

- Internal meetings (purple): `#A78BFA` bg, `#8B5CF6` border
- Task deadlines (orange): `#FBBF24` bg, `#F59E0B` border
- Reminders (green): `#34D399` bg, `#10B981` border
