**Parent**: [OPS-338](ops-338.md) | **Phase**: 6 (Admin & Settings)

## Problem

Reports page needs multiple report types with date range filtering and export.

## Approach

- Create ReportsPage with report type selector
- Implement CaseMetricsReport, TimeReport, BillingReport, AIUsageReport
- Add date range picker and export buttons
