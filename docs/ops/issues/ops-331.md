# OPS-331: Migrate Sidebar to Linear Design

**Type**: UI Enhancement | **Complexity**: M | **Platform**: Desktop
**Status**: Open
**Epic**: Linear UI Migration (OPS-331 to OPS-337)

## Problem

The main Sidebar navigation uses older styling patterns. It needs to adopt Linear design tokens for colors, typography, hover states, and active indicators to match the migrated components.

## Approach

- Apply Linear background colors (`linear-bg-primary`, `linear-bg-secondary`)
- Update nav items to use `ListItem` pattern with hover expansion
- Add Linear accent color for active states with left border indicator
- Use 11px uppercase section headers
- Update icons to consistent sizing

## Done When

- [x] Sidebar uses Linear color tokens throughout
- [x] Nav items have Linear hover/active states
- [x] Section headers use Linear typography
- [x] Collapsed state maintains Linear styling
- [x] Dark mode renders correctly
