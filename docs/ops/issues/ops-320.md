## Goal

Restyle mobile-specific components with Linear dark mode design.

## Tasks

### 1. MobileTabBar

- Dark background (`#111113`)
- Active tab with accent color
- Safe area padding for home indicator
- Touch targets minimum 44x44px

### 2. MobileDrawer

- Dark slide-in panel
- User section with gradient avatar
- Navigation items with icons
- Settings section

### 3. MobileHome

- Dark masthead with date/greeting
- Briefing section styling
- Stats cards horizontal scroll
- Pull-to-refresh indicator

### 4. BriefRow / FeedSection

- Dark row styling
- Icon backgrounds with status colors
- Touch active states (scale 0.98)
- Swipe gestures

### 5. AssistantFAB

- Gradient accent background
- Glow shadow effect
- Touch feedback

## CSS Patterns

```css
/* Mobile tab bar */
.tab-bar {
  background: #111113;
  border-top: 1px solid rgba(255, 255, 255, 0.06);
  padding-bottom: env(safe-area-inset-bottom);
}

/* Active tab */
.tab-active {
  color: #5e6ad2;
}

/* Touch active state */
.touchable:active {
  transform: scale(0.98);
  opacity: 0.9;
}

/* FAB glow */
.fab {
  background: linear-gradient(135deg, #5e6ad2, #6b76dc);
  box-shadow: 0 0 40px rgba(94, 106, 210, 0.3);
}
```
