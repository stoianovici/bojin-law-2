# OPS-369: Keyboard Shortcuts System

**Type**: Feature | **Priority**: P2-Medium | **Platform**: Desktop | **Status**: Done
**Parent**: [OPS-338](ops-338.md) | **Phase**: 7 (Polish)
**Blocked by**: ~~OPS-354~~ (resolved)

## Problem

Need global keyboard shortcuts for power users: ⌘K, ⌘N, ⌘T, ⌘L, ⌘J, ⌘G, Escape.

## Approach

- Create useKeyboardShortcuts hook
- Implement global shortcuts with Command Palette integration
- Add modal shortcuts (⌘Enter submit, ↑↓ navigation)
- Add shortcut hints in UI where relevant

## Done When

- [x] Global shortcuts: ⌘K (palette), ⌘N (new case), ⌘T (new task), ⌘L (log time), ⌘J (AI), ⌘G (go to case)
- [x] Escape closes modals
- [x] Modal shortcuts: ⌘Enter submit (documented), ↑↓ navigation (in command palette)
- [x] Shortcut hints shown in ShortcutReference panel (⌘/)

## Shortcut Map

| Shortcut | Action                    |
| -------- | ------------------------- |
| ⌘K       | Open Command Palette      |
| ⌘N       | New case                  |
| ⌘T       | New task                  |
| ⌘L       | Log time                  |
| ⌘J       | AI assistant              |
| ⌘G       | Go to case                |
| Escape   | Close modal/panel         |
| ⌘Enter   | Submit form (in modals)   |
| ↑↓       | Navigate items (in lists) |
| Enter    | Select item               |
