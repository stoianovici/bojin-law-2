# OPS-333: Migrate Dashboard Widget System to Linear Design

**Type**: UI Enhancement | **Complexity**: M | **Platform**: Desktop
**Status**: Closed
**Epic**: Linear UI Migration (OPS-331 to OPS-337)

## Problem

Dashboard components (PartnerDashboard, AssociateDashboard, WidgetContainer) are only 33% migrated and use older widget patterns that don't match the Linear design system.

## Approach

- Refactor WidgetContainer to use Linear `Widget` component pattern
- Update dashboard grids to use `WidgetGrid` layout
- Apply Linear card styling with hover elevation
- Migrate metric displays to use `MetricCard` components
- Update MorningBriefing to use `BriefingCard` pattern

- [x] All dashboard cards have Linear hover states
