# OPS-148: Skeleton Shimmer Loading Pattern

**Type**: Feature
**Priority**: P2-Medium
**Status**: Closed
**Complexity**: S
**Parallel**: Blocked by OPS-143

## Problem

Current skeleton loading uses basic `animate-pulse` (opacity pulse). The shimmer effect (gradient moving across) is more polished and is the industry standard for loading states.

## Solution

1. Create shimmer gradient utility class using `animate-shimmer`
2. Replace `animate-pulse` with shimmer pattern in key loading states

## Implementation

### Shimmer Pattern

```tsx
// Reusable skeleton with shimmer
<div
  className={clsx(
    'animate-shimmer rounded',
    'bg-gradient-to-r from-gray-200 via-gray-100 to-gray-200',
    'bg-[length:200%_100%]'
  )}
/>
```

### Apply to Components

Replace `animate-pulse` with shimmer in:

- MorningBriefing skeleton
- DocumentList skeleton
- ThreadList skeleton
