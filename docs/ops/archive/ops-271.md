# OPS-271: Team Activity Page Foundation

**Type**: Feature
**Priority**: P1-High
**Status**: Closed
**Blocked By**: None

## Problem

No dedicated page for partner team oversight and timesheet generation. Partners need to see what the team worked on and prepare timesheets for client invoicing, but must currently dig through multiple views to compile this information.

## Solution

Create `/activitate-echipa` page with shell structure, routing, shared filters, and mode toggle between "Vizualizare" (view mode) and "Fișă de pontaj" (timesheet mode).

## Implementation

1. Create `apps/web/src/app/activitate-echipa/page.tsx` - page shell with two-column layout (filters sidebar + content area)
2. Create `TimesheetFilters.tsx` - filters panel with:
   - Case picker (reuse existing `CaseCombobox` component)
   - Team member multi-select filter
   - Period selector (month picker with presets: Astăzi, Săptămâna, Luna)
3. Create `ModeToggle.tsx` - toggle between Vizualizare and Fișă de pontaj using Radix Toggle
4. Add page to sidebar navigation with role check (Partner/BusinessOwner only)
5. Create `useTeamActivity.ts` hook skeleton for data fetching
