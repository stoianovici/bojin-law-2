# OPS-219: Expand CaseActorRole Enum

**Type**: Feature | **Priority**: P2-Medium | **Complexity**: S

## Problem

Only 5 actor roles exist (Client, OpposingParty, OpposingCounsel, Witness, Expert). Romanian legal practice needs more roles for proper case party categorization.

## Solution

Extend CaseActorRole enum with 8 additional common Romanian legal roles. Update frontend label arrays with Romanian translations.

## New Enum Values

| Code                  | Romanian Label        |
| --------------------- | --------------------- |
| `Intervenient`        | Intervenient          |
| `Mandatar`            | Mandatar              |
| `Court`               | Instanță              |
| `Prosecutor`          | Procuror              |
| `Bailiff`             | Executor Judecătoresc |
| `Notary`              | Notar                 |
| `LegalRepresentative` | Reprezentant Legal    |
| `Other`               | Altele                |

## Implementation

1. Add enum values to CaseActorRole in schema.prisma
2. Create Prisma migration
3. Update ACTOR_ROLES array in new case page with Romanian labels
4. Update ContactsSection role display labels
