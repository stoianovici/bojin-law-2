# OPS-149: Tab & Panel Transition Animations

**Type**: Feature
**Priority**: P3-Low
**Status**: Open
**Complexity**: M
**Parallel**: Blocked by OPS-144, OPS-146

## Problem

Tab content switches instantly without transition. Side panels appear/disappear abruptly. Multi-step modals jump between steps. These instant changes feel jarring in an otherwise polished UI.

## Solution

1. Add cross-fade animation to tab content panels
2. Use `AnimatedPanel` for sliding sidebars/panels
3. Add slide transition to `CreateCaseModal` wizard steps

## Implementation

### Tab Content Cross-Fade

```tsx
import { motion } from 'framer-motion';

const TabsContent = forwardRef(({ className, children, ...props }, ref) => (
  <TabsPrimitive.Content ref={ref} className={clsx('mt-2', className)} {...props}>
    <motion.div
      initial={{ opacity: 0, y: 10 }}
      animate={{ opacity: 1, y: 0 }}
      exit={{ opacity: 0, y: -10 }}
      transition={{ duration: 0.15 }}
    >
      {children}
    </motion.div>
  </TabsPrimitive.Content>
));
```

### CreateCaseModal Step Transitions

```tsx
import { AnimatePresence, motion } from 'framer-motion';

const stepVariants = {
  enter: (direction: number) => ({ x: direction > 0 ? 20 : -20, opacity: 0 }),
  center: { x: 0, opacity: 1 },
  exit: (direction: number) => ({ x: direction < 0 ? 20 : -20, opacity: 0 }),
};

<AnimatePresence mode="wait" custom={direction}>
  <motion.div
    key={currentStep}
    custom={direction}
    variants={stepVariants}
    initial="enter"
    animate="center"
    exit="exit"
    transition={{ duration: 0.2 }}
  >
    {renderStepContent()}
  </motion.div>
</AnimatePresence>;
```
