# OPS-209: DraftModeToolbar Component

**Type**: Feature | **Priority**: P2-Medium | **Complexity**: S
**Status**: Closed

## Problem

Draft mode needs floating save/cancel toolbar with validation feedback.

## Approach

Sticky bottom toolbar with validation state, save button (disabled if invalid), and cancel button. Shows required field hints.

```tsx
interface DraftModeToolbarProps {
  onSave: () => Promise<void>;
  onCancel: () => void;
  isDirty: boolean;
  isValid: boolean;
  isSaving: boolean;
  validationErrors?: string[];
  showApprovalNotice?: boolean; // For Associates
}

function DraftModeToolbar({
  onSave,
  onCancel,
  isDirty,
  isValid,
  isSaving,
  validationErrors,
  showApprovalNotice,
}: DraftModeToolbarProps) {
  return (
    <div className="fixed bottom-0 left-0 right-0 bg-white border-t shadow-lg p-4">
      {showApprovalNotice && (
        <div className="text-sm text-amber-600 mb-2">⚠️ Dosarul va fi trimis spre aprobare</div>
      )}
      {validationErrors?.length > 0 && (
        <div className="text-sm text-red-500 mb-2">{validationErrors.join(', ')}</div>
      )}
      <div className="flex justify-end gap-3">
        <Button variant="ghost" onClick={onCancel}>
          Anulează
        </Button>
        <Button onClick={onSave} disabled={!isValid || isSaving} loading={isSaving}>
          Creează Dosar
        </Button>
      </div>
    </div>
  );
}
```
