# OPS-218: Court Case Number Field in Case UI

**Type**: Feature | **Priority**: P2-Medium | **Complexity**: S

## Problem

The `caseNumber` database field exists but isn't exposed in the UI. Users store court case numbers in the references section instead of a dedicated field.

## Solution

Surface `caseNumber` as "Nr. Dosar Instanță" in CaseDetailsSection and new case page, positioned prominently below title. Use existing InlineEditField pattern.

## Implementation

1. Add `caseNumber` to `UpdateCaseInput` in case.graphql
2. Add InlineEditField for "Nr. Dosar Instanță" in CaseDetailsSection (below title)
3. Add `caseNumber` field to useDraftCase hook draft state
4. Add field to `/cases/new` page form
