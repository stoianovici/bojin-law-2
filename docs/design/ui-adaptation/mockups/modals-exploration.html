2. Form Modal

Create/Edit Entity Form Modal
Sarcină nouă
⌘ + Enter salvează
Use for: Creating or editing entities (tasks, cases, time entries). Has header with title, scrollable body, sticky footer with keyboard hint.
Quick Input Compact Form

Adaugă notă

Use for: Simple single-field inputs (notes, comments, rename). No header bar needed, compact layout.
