Empty / Loading / Error States

Patterns for handling non-ideal UI states

1. Empty States

Page Empty Full Page

Niciun document

Nu ați adăugat încă documente la acest dosar. Încărcați primul document pentru a începe.

Icon + Title + Description + CTA. Centered on page.
Widget Empty Card/Widget

Totul gata!

Nu aveți sarcini urgente pentru azi. Bucurați-vă de o zi productivă!

Neutral icon (gray). Positive message. No CTA needed.
Inline Empty Section/List
Nu există comentarii la această sarcină
Compact. For sections within a page. Minimal styling.
Search No Results Search

Niciun rezultat

Nu am găsit rezultate pentru "contract vânzare". Încercați alți termeni de căutare.

Shows search term. Secondary action to clear search.

2. Loading States

Spinner Page/Section
Se încarcă...
3 sizes: sm (16px), md (32px), lg (48px). Optional text below.
Skeleton Card Card Loading
Mirrors content structure. Subtle shimmer animation.
Progress Bar Upload/Process
65% - Se încarcă document...
Se procesează...
Determinate (with %) or indeterminate (sliding).
Overlay Loading In-place Refresh
Se actualizează...
Content dimmed, spinner on top. For refreshing existing content.

3. Error States

404 Not Found Full Page
404

Pagină negăsită

Pagina pe care o căutați nu există sau a fost mutată.

Error icon + code + title + description + action.
API Error Section

Eroare la încărcare

Nu am putut încărca datele. Verificați conexiunea și încercați din nou.

Warning icon (orange). Retry action.
Inline Error Form/Section
Eroare la salvare
Câmpul "Client" este obligatoriu. Vă rugăm să completați toate câmpurile marcate.
Compact. For form validation, section errors.
Toast Error Notification
Sincronizare eșuată
Nu s-a putut conecta la Microsoft 365.
Bottom-right. Auto-dismiss or with action.

When to Use What

State Pattern When to Use
Empty (Page) Icon + Title + CTA First-time user, no data yet, actionable
Empty (Widget) Icon + Title only Section has no data, neutral/positive message
Empty (Inline) Small icon + text Comments, activity feed, compact spaces
Empty (Search) Icon + message + clear No search results, show query, offer to clear
Loading (Initial) Skeleton or Spinner Page/section loading for first time
Loading (Refresh) Overlay + Spinner Refreshing existing content
Loading (Upload) Progress bar File upload, long process with known progress
Error (Page) Full error state 404, 403, 500 - entire page failed
Error (Section) Centered error API call failed, can retry
Error (Inline) Banner style Form validation, section-level errors
Error (Toast) Toast notification Background action failed, non-blocking
