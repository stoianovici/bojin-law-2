Card Component

Default Variant

Card Default

Subtitlu opțional

Conținutul cardului poate include text, imagini, sau alte componente. Utilizează border subtil fără shadow.

Elevated Variant

Card Elevated

Cu shadow subtil

Varianta elevated folosește shadow-sm în loc de shadow-lg. Shadow-ul crește subtil la hover, dar rămâne discret.

Outlined Variant

Card Outlined

Cu border mai vizibil

Varianta outlined are border-width: 2px și folosește border-linear-border în loc de border-linear-border-subtle.

Simple Card (Body Only)

Un card simplu poate avea doar conținut în body, fără header sau footer. Ideal pentru blocuri de text sau widget-uri simple.
